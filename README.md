# hubot-choice

A hubot script that choice one

See [`src/choice.coffee`](src/choice.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-choice --save`

Then add **hubot-choice** to your `external-scripts.json`:

```json
[
  "hubot-choice"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
